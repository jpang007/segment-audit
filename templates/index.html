📚 How to Get Schema CSVs from Segment

  1. In Segment, navigate to Connections → Sources
  2. Select the source you want to analyze (e.g., JavaScript, iOS, Android)
  3. Click the Schema tab
  4. Click Download CSV button (top right)
  5. Repeat for each source you want to include in your unified tracking plan

Single Source:

Volume-based recommendations for one data source (✅ Include, 🟡 Review, 🔴 Discard)
