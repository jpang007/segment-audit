📚 Quick Help

Get JWT Token:

Login to Segment → DevTools → Application → Cookies → auth_token

Token Security:

Your token is used only to call Segment Gateway APIs for this audit. It is not sent to third-party services unless experimental AI features are enabled. Audit data is stored temporarily for the session and can be deleted after export.

Token Expiry:

JWT tokens typically expire after 7 days - just grab a fresh one

Test Environment:

This runs on port 5003 for testing Gateway API capabilities
